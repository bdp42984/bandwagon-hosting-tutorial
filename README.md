## 二、新手需要知道的几个基本概念

在买搬瓦工之前，这几个词你应该先弄明白，买的时候不会懵：

**KVM**：虚拟化技术类型，搬瓦工全线都是 KVM，独立内核，性能稳定，支持 BBR 加速等各种内核参数。

**CN2 GIA / CN2 GT**：两种网络线路。CN2 GIA 是电信高端线路，全程走优化路由，晚高峰也稳；CN2 GT 是普通版，日常够用，晚高峰可能有点卡。价格上 GIA 比 GT 贵一截。

**CMIN2 / 软银**：分别是移动和联通/日本软银的优化线路，不同运营商的用户选线路讲究不一样。

**机房（DC）**：数据中心的位置，比如 DC6、DC9 都是洛杉矶的不同机房，走的线路也有差异。

**KiwiVM**：搬瓦工自己开发的 VPS 管理面板，购买之后所有操作——开关机、重装系统、看 IP 端口、换机房——都在这里搞定。

## 七、购买后怎么用：KiwiVM 面板 + SSH 连接完整教程

买完 VPS 只是第一步，接下来要连上去才能用。

### 第一步：进入 KiwiVM 控制面板

登录搬瓦工账号后台 → 点击 Services → My Services → 找到你的 VPS → 点击 **KiwiVM Control Panel** 进入。

进入 KiwiVM 之后，首页就能看到：
- **Public IP address**：你 VPS 的 IP 地址
- **SSH Port**：SSH 端口（搬瓦工是随机端口，不是默认的 22）

### 第二步：获取 root 密码

新买的 VPS 需要自己重置密码。点击左侧菜单的 **Root password modification** → **Generate and set new root password**，生成后记下来（只显示一次！）。

注意：重置密码时 VPS 必须是开机状态。

### 第三步：选一个 SSH 工具连上去

**Windows 用户**：推荐 Xshell（免费版够用）或者 Windows 自带的 PowerShell/终端。

**macOS / Linux 用户**：直接用系统自带终端，输入：

bash
ssh root@你的IP地址 -p 你的SSH端口


比如：

bash
ssh root@123.456.789.10 -p 25639


回车后输入刚才设置的密码，看到命令提示符就说明连上了。

恭喜，你现在可以在上面随便折腾了。

## 八、连上 SSH 之后，新手可以做什么？

几个常见的入门操作：

**更新系统软件包**（第一次连上建议先跑一遍）：
bash
apt update && apt upgrade -y   # Debian/Ubuntu


**安装宝塔面板**（图形化建站工具，新手友好）：
bash
wget -O install.sh https://download.bt.cn/install/install_lts.sh && bash install.sh


**安装 BBR 加速**（提升网络速度的内核优化）：
bash
echo "net.core.default_qdisc=fq" >> /etc/sysctl.conf
echo "net.ipv4.tcp_congestion_control=bbr" >> /etc/sysctl.conf
sysctl -p


**重装操作系统**：不满意当前系统，在 KiwiVM 面板里找 **Reinstall OS**，选 Ubuntu 24.04 LTS 一键重装，2-3 分钟完成。

**切换机房**：如果发现当前机房速度不好，在 KiwiVM 里找 **Migration**，选目标机房点击 Migrate，1-2 分钟迁移完成，IP 会变，数据不丢失（仅限支持迁移的套餐，如 CN2 GIA-E）。

## 九、新手常踩的几个坑

**1. 买完找不到 VPS 在哪**：登录搬瓦工后台 → Services → My Services，在这里找到已购买的 VPS。

**2. SSH 连不上**：先检查 IP 和端口是否填对（端口不是 22），再检查密码是否在 KiwiVM 重置过，还不行可以尝试在 KiwiVM 里用在线 SSH 工具（Root shell – basic）测试。

**3. 密码忘了/过期**：去 KiwiVM → Root password modification 重置，非常简单。

**4. 买的时候没用优惠码**：搬瓦工的优惠码是循环折扣，买了之后每次续费都自动享受折扣，所以买的时候如果有可用的优惠码，一定要用。截至2026年4月，最新优惠码情况请出发前在搬瓦工官方渠道确认。
